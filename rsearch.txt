Branching
--------------------
1- Be in Master
     git checkout master

2-Create branch
     git branch branch_name

3-Switch to new branch
      git checkout branch_name

4- after changes (in IDE) check status and commit
      git status
      git commit -am "comments"  (or do in VS)
      **(extra to check commit logs) git log

Mongo
-----
Installation:  1- Run MongoDb-win32 installer
	       2- create directory C:\data\db
               3- In cmd shell navigate to install directory: C:\Program Files\MongoDB\Server\3.2\bin
               4- run the Db engine mongod.exe. That's it!
               EXTRA. To manged with UI tool RoboMongo(now Robo 3T)
                      1- Install RoboMongo and connect to localhosts, create database etc..
